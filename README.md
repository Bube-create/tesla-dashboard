# tesla-dashboard
A tesla model 3 dashboard simulator, hope you have fun playing with it

features: 
- international radio.
- sketchpad easter egg .
- map location directions.
- and other cool stuff.


- Live Preview: https://tesla-dashboard-five.vercel.app/
